## Local setup

- Fork this repo (top right corner)
- Clone your forked repo using the [`git clone`](https://harshkapadia2.github.io/git_basics/#_git_clone) command.
- Make your changes to the repo!
- Push the code to your forked repo.
- Make a PR!
